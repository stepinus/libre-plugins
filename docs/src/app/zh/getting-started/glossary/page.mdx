# 术语表

LibreFang 相关术语和概念定义。


---

## A

### Agent
自主运行的软件实体，可以执行任务、做出决策并与用户或其他系统交互。

### API (Application Programming Interface)
应用程序编程接口，允许外部程序与 LibreFang 通信。

### A2A (Agent to Agent)
Agent 之间通信的协议标准。

---

## B

### Bearer Token
用于 API 认证的令牌格式。

### Benchmark
性能测试，用于评估系统速度、吞吐量和响应时间。

---

## C

### Capability
Agent 被授权执行的操作范围，如使用特定工具或访问特定资源。

### Channel
消息平台适配器，允许 Agent 通过不同平台与用户通信。

### CLI (Command Line Interface)
命令行界面，通过终端与 LibreFang 交互。

### Compaction
会话压缩，减少历史消息以控制 token 使用。

---

## D

### Daemon
后台运行的 LibreFang 守护进程。

### DAG (Directed Acyclic Graph)
有向无环图，工作流的执行模型。

---

## E

### Ed25519
非对称加密算法，用于签名和验证。

### Environment Variable
环境变量，存储配置信息的键值对。

---

## F

### FangHub
LibreFang 技能市场。

### Fallback Provider
后备 LLM 提供商，当主提供商失败时使用。

---

## G

### GCRA (Generic Cell Rate Algorithm)
通用单元速率算法，用于速率限制。

### Groq
LLM 提供商之一，以快速推理著称。

---

## H

### Hand
LibreFang 的自主能力包，可以独立运行和执行任务。

### HTTP Headers
HTTP 请求/响应的元数据。

---

## K

### Kernel
LibreFang 核心组件，负责协调所有子系统。

### Knowledge Graph
知识图谱，存储实体和关系的结构化数据。

---

## L

### LLM (Large Language Model)
大型语言模型，如 Claude、GPT 等。

### LLM Driver
LibreFang 与不同 LLM 提供商交互的适配层。

---

## M

### MCP (Model Context Protocol)
模型上下文协议，连接 LLM 与外部工具的标准化方式。

### Memory
LibreFang 的持久化存储系统，包括会话、向量搜索和知识图谱。

### Merkle Hash Chain
默克尔哈希链，用于审计追踪的数据结构。

### Model Catalog
模型目录，包含所有可用模型和提供商信息。

---

## O

### OFP (OpenFang Protocol)
LibreFang 点对点通信协议。

### OpenAI Compatible API
与 OpenAI API 格式兼容的接口。

---

## P

### Prompt Injection
提示注入，安全威胁类型，通过恶意输入操纵 AI 行为。

### Provider
LLM 提供商，如 Anthropic、OpenAI、Groq 等。

---

## R

### RBAC (Role-Based Access Control)
基于角色的访问控制。

### REST API
表述性状态传递 API，Web 服务标准架构风格。

---

## S

### SSE (Server-Sent Events)
服务器发送事件，用于服务器到客户端的单向流式传输。

### Skill
可插拔的工具包，扩展 Agent 能力。

### SSRF (Server-Side Request Forgery)
服务器端请求伪造，安全威胁类型。

### Stable Mode
Agent 运行模式，固定模型和技能注册表。

### System Prompt
系统提示，定义 Agent 的行为和人格。

---

## T

### Taint Tracking
污点追踪，安全机制，跟踪敏感信息流动。

### TOML
Tom's Obvious, Minimal Language，配置文件格式。

### Tool
Agent 可以调用的功能，如文件读取、网络请求等。

### Trigger
触发工作流执行的事件源。

---

## V

### Vector Embedding
向量嵌入，将文本转换为数值向量用于语义搜索。

### Vault
凭证保险库，安全存储敏感信息。

---

## W

### WASM (WebAssembly)
WebAssembly，安全的沙箱运行环境。

### WebSocket
全双工通信协议，支持实时双向交互。

### Workflow
工作流，由触发器和步骤组成的自动化流程。

---

## Z

### Zeroizing
内存零化，安全擦除敏感数据。
